O que são tags HTML?

As tags são usadas para informar ao navegador a estrutura do site. Ou seja, quando se escreve um código em HTML, as tags serão interpretadas pelo navegador, produzindo assim a estrutura e o conteúdo visual da página. A principal característica das tags é estarem sempre dentro dos sinais de “maior que” e “menor que”, ou seja: <>. As tags que precisam de fechamento possuem a sintaxe , já as que não precisam de fechamento possuem como estrutura .

Qual a estrutura básica de um documento HTML?

Para um documento HTML ser válido precisa seguir obrigatoriamente uma estrutura composta pelas tags html, head e body e a instrução !DOCTYPE. Esta estrutura está informando aos navegadores como eles devem considerar e interpretar estas tags

  • HTML - Essa tag diz ao navegador que o que estiver dentro dela deve ser encarado como elementos de um código HTML.
  • Doctype - O doctype não é uma tag HTML, mas uma instrução para o navegador e outros programas que podem ler seu site.
  • Head - Contém toda as configurações do site, é como se fosse os pensamentos do site.
  • Body - é nessa tag que todos os elementos visíveis do seu site devem ser inseridos.

Tags de Textos

As tags de texto definem textos, estilos de fonte, parágrafos, spans, quebras de linhas, etc.

  • <p></p> - Utilizada para criação de parágrafos.
  • <span></span> - Pode ser bem parecida com a tag de parágrafos, mas ela costumam ser utilizadas apenas para pequenas informações, como legendas de um formulário, legendas de uma imagem, etc...
  • <hr> - Criação de uma linha horizontal
  • <br> - Utilizada para quebra de linha
  • <em> - utilizada para por o texto em italico
  • <mark> - Utilizada para marcar um texto
  • <strong> - Utilizada para deixar o texto em negrito

Tags de links

A HTML usa a tag <a></a> (âncora) para criar um vínculo (link) com outro documento. Uma âncora pode apontar para qualquer recurso na Web: uma página em HTML, uma imagem, um arquivo de som, um filme, etc. Além do atributo href, também podemos incluir o atributo target dentro da tag <a></a>. Esse atributo informa ao navegador como o redirecionamento deverá ocorrer, abrindo a página na mesma janela/aba do navegador ou abrindo uma nova janela/aba.

Os atributos são:

  • _blank: abre a página em uma nova janela/aba.
  • _self: abre a página na mesma janela.
  • _parent: abre a página na mesma janela do link.
  • _top: cancela todos os demais frames e abre a nova página no mesmo navegador.
